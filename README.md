# Парсер образовательных программ для проекта «Ресоциализация СВО»

## Описание
Скрипт для сбора информации об образовательных программах государственных ВУЗов и ССУЗов РФ с бюджетными местами на 2024/2025 учебный год.

## Требования к окружению

- Python 3.8+
- Библиотеки: см. файл `requirements.txt`

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Подготовка

1. Убедитесь, что файл `target_list.json` находится в корневой директории проекта
2. Проверьте доступ к сайтам-источникам (vuzopedia.ru, postupi.online)
3. При необходимости настройте прокси в случае блокировок

## Запуск

```bash
python parser_educational_programs.py
```

## Параметры командной строки

Настройки можно изменить в коде:
- `min_programs`: минимальное количество программ для сбора (по умолчанию 50)
- `output_file`: имя файла для сохранения результатов

## Результат работы

- `educational_programs_2024.csv` - основной файл с результатами
- `educational_programs_2024_report.txt` - отчет о сборе данных
- `parser.log` - лог выполнения операций

## Структура CSV-файла

| Поле | Описание |
|------|----------|
| id | Уникальный идентификатор записи |
| macrogroup_id | ID макрогруппы (1-10) |
| macrogroup_name | Название макрогруппы |
| education_level | Уровень образования (Высшее/СПО) |
| fgos_code | Код специальности по ФГОС |
| program_name | Название образовательной программы |
| institution_name | Название учебного заведения |
| region | Регион расположения учреждения |
| budget_seats | Количество бюджетных мест |
| url | URL страницы программы |

## Возможные проблемы и решения

1. **Сайты изменили структуру** - обновите селекторы в соответствующих методах парсера
2. **Блокировка IP** - используйте прокси или увеличьте задержки между запросами
3. **Недостаточно данных** - добавьте дополнительные источники в код
4. **Медленная работа** - проверьте соединение и при необходимости оптимизируйте с помощью асинхронных запросов

## Источники данных

1. Vuzopedia.ru (приоритетный источник)
2. Postupi.online (резервный источник)
3. Ucheba.ru (в случае необходимости)

## Статистика сбора

Скрипт автоматически формирует отчет с:
- Общим количеством собранных программ
- Распределением по источникам
- Распределением по макрогруппам
- Распределением по регионам
- Статистикой по бюджетным местам
