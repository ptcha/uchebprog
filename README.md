# Учебный проект

Этот репозиторий содержит веб-приложение "Портал образовательных программ РГСУ".

## Функциональность

Сайт автоматически извлекает данные из Google таблицы и отображает их в виде карточек:
- Источник данных: https://docs.google.com/spreadsheets/d/12GNjcomxxU4NbXKwQ1Jq_eG7M7PLUlDlKAnBO4qwu5g/edit?gid=0#gid=0
- Данные загружаются при открытии страницы и отображаются в виде интерактивного каталога
- Поддерживается фильтрация по различным критериям (уровень образования, бюджетные места, федеральный округ и т.д.)

## Структура проекта

- `index.html` - главная страница приложения
- `tailwind.css` - стили Tailwind CSS
- `fonts.css` - стили шрифтов Google Fonts
- `lucide.js` - библиотека иконок Lucide
- `marked.min.js` - парсер Markdown
- `papaparse.min.js` - парсер CSV
- `20240520114700!РГСУ_логотип.png` - логотип РГСУ
