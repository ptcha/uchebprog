## Как работает реализация

### 1. Извлечение данных
- В строках 2686-2687 файла `index.html` указан ID таблицы: `12GNjcomxxU4NbXKwQ1Jq_eG7M7PLUlDlKAnBO4qwu5g`
- Сайт формирует URL для экспорта таблицы в формате CSV: `https://docs.google.com/spreadsheets/d/${SHEET_ID}/export?format=csv&gid=0`
- Используется функция `loadData()` (строки 2689-2748) для асинхронной загрузки данных

### 2. Парсинг данных
- Используется библиотека PapaParse (строка 2705) для парсинга CSV
- Данные маппятся в структуру приложения с поддержкой как английских, так и русских заголовков столбцов (строки 2714-2731)
- Поддерживаются следующие поля:
  - `Program_Name` или `Название` или `title` → название программы
  - `FGOS_Code` или `Код` → код специальности
  - `FGOS_Name` или `Специальность` → название специальности
  - `Institution_Name` или `ВУЗ` → учебное заведение
  - `Level` или `Уровень` → уровень образования
  - `Description` или `Описание` → описание программы
  - и другие поля

### 3. Отображение карточек
- Карточки отображаются в контейнере с ID `catalogGrid` (строка 282)
- Используется функция `renderCards()` (строки 2826-2879) для генерации HTML
- Каждая карточка содержит:
  - Уровень образования (Бакалавриат/Колледж)
  - Федеральный округ
  - Название программы
  - Описание
  - Метки (бюджет, квоты и т.д.)
  - Учебное заведение и срок обучения

### 4. Фильтрация
- Реализована фильтрация по различным критериям (строки 2802-2824)
- Поддерживается поиск по названию, коду, тегам и учебному заведению
- Фильтрация по уровню образования (ВО/СПО)
- Фильтрация по бюджетным местам
- Фильтрация по федеральному округу

## Использование
Просто откройте `index.html` в браузере - карточки будут автоматически заполнены данными из Google таблицы. Если таблица недоступна, используется резервная база данных.

## Технические детали
- Для работы используется библиотека PapaParse (встроена в проект)
- Страница автоматически загружает данные при инициализации (строка 2779)
- В случае ошибки загрузки данных из таблицы используется резервная база данных (строки 2742-2743)
