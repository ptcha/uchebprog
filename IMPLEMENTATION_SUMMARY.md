# Реализация парсера образовательных программ для проекта «Ресоциализация СВО»

## Обзор проекта

Реализован полный комплекс для сбора информации об образовательных программах государственных ВУЗов и ССУЗов РФ с бюджетными местами на 2024/2025 учебный год.

## Структура проекта

```
/workspace/
├── parser_educational_programs.py    # Основной парсер
├── target_list.json                 # Целевой список специальностей
├── requirements.txt                 # Зависимости
├── README.md                       # Инструкция по запуску
├── test_parser.py                  # Тестовый скрипт
├── educational_programs_2024.csv   # Результаты парсинга
├── educational_programs_2024_report.txt  # Отчет о сборе
└── IMPLEMENTATION_SUMMARY.md       # Настоящий документ
```

## Реализованные функции

### 1. Основной парсер (parser_educational_programs.py)

- **Архитектура**: ООП-класс `EducationalProgramParser`
- **Источники данных**: Vuzopedia.ru (приоритет), Postupi.online (резерв)
- **Фильтрация**: 
  - Только государственные учреждения
  - Наличие бюджетных мест
  - Соответствие целевым специальностям
- **Структура вывода**: 
  - id, macrogroup_id, macrogroup_name, education_level, fgos_code
  - program_name, institution_name, region, budget_seats, url

## Структура CSV-файла

Файл `educational_programs_2024.csv` содержит:
- **id**: уникальный идентификатор записи
- **macrogroup_id**: ID макрогруппы (1-10)
- **macrogroup_name**: название макрогруппы
- **education_level**: уровень образования (Высшее/СПО)
- **fgos_code**: код специальности по ФГОС
- **program_name**: название образовательной программы
- **institution_name**: название учебного заведения
- **region**: регион расположения учреждения
- **budget_seats**: количество бюджетных мест
- **url**: URL страницы программы

## Отчет о сборе

Файл `educational_programs_2024_report.txt` содержит:
- Общее количество собранных программ
- Распределение по источникам
- Распределение по макрогруппам
- Распределение по уровням образования
- Распределение по регионам
- Статистика по бюджетным местам

## Инструкция по запуску

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите парсер: `python parser_educational_programs.py`
3. Результаты будут сохранены в `educational_programs_2024.csv`
